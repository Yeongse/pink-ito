# Product Overview

「ピンク Ito」は、大人向けの数字当て協力型パーティゲームアプリ。1台のスマートフォンを回し見しながら、下ネタをテーマにしたお題に対して各自の数字を表現し、全員で正しい順番を当てる。

## Core Capabilities

1. **オフライン完結型ゲームプレイ** - インターネット接続不要、1台のデバイスで2-10人がプレイ可能
2. **回し見式の数字配布** - 各プレイヤーに順番に秘密の数字（1-100）を配布
3. **協力型順序当て** - 話し合いとドラッグ&ドロップで正解を目指す
4. **繰り返しプレイ対応** - セッション中のお題重複回避と自動リセット

## Target Use Cases

- 飲み会・パーティでの大人向けアイスブレイク
- 友人同士の親密なゲームタイム
- 合コン・二次会でのエンターテインメント

## Value Proposition

オリジナルの「Ito」ゲームを下ネタ特化にアレンジし、スマホ1台で手軽にプレイできる。ネオンピンクを基調としたスタイリッシュなUIで、下品になりすぎないセンスの良い大人の遊びを提供。
