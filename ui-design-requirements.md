# ピンク Ito - UIデザイン要件書

## デザインコンセプト

### 全体ビジョン
**「ネオン×アダルトバー」** - 大人のパーティゲームにふさわしい、夜の繁華街を思わせるセクシーで洗練されたビジュアル。安っぽさを排除し、スタイリッシュな大人の遊び心を表現する。

### トーン＆ムード
- **Luxe Neon（ラグジュアリー × ネオン）**: 高級感とネオンサインの妖艶さの融合
- **Playful Sophistication**: 下ネタというテーマをエレガントに昇華
- **Intimate & Inviting**: パーティの親密感、秘密の共有感

### 差別化ポイント
「下ネタゲーム」でありながら下品にならない、**洗練されたアダルトな雰囲気**。記憶に残るのは、ダークな背景に浮かぶネオンピンクのグロー効果と、滑らかなアニメーション。

### フォントファミリー

**Zen Kaku Gothic Antique** または **Noto Sans JP** (Black weight)
- お題表示、数字、重要な見出しに使用
- ウェイト: 900 (Black)

**Noto Sans JP** (Regular/Medium)

## 画面別デザイン仕様

### 1. タイトル画面

#### レイアウト
- 中央配置、縦スタック
- 背景: ダークグラデーション + 微細なノイズテクスチャ
- 装飾: 画面端にぼんやりとしたネオンピンクのグロー

#### 要素
- **ロゴ**: 「ピンク Ito」をネオンサイン風に表現
  - テキストに複数レイヤーのグロー効果
  - 微細な点滅アニメーション（ネオン管風）
- **サブタイトル**: 「大人の数字当てゲーム」（控えめに）
- **スタートボタン**: プライマリボタン、パルスアニメーション

#### アニメーション
- ロゴ: フェードイン + スケールアップ（0.8 → 1.0）
- ネオングロー: 2秒周期でゆっくり明滅
- ボタン: 3秒周期でパルス

### 2. プレイヤー設定画面

#### レイアウト
- 上部: 画面タイトル「プレイヤー設定」
- 中央: プレイヤー数スライダー + 名前入力リスト
- 下部: 「ゲーム開始」ボタン

#### プレイヤー数選択
- 横スライダー（2-10人）
- 選択数字をネオンピンクで大きく表示
- 両端に「-」「+」ボタン

#### 名前入力リスト
- スクロール可能なリスト
- 各行: 番号 + 入力フィールド
- 入力完了行は軽いチェックマーク表示
- アニメーション: 行追加時にスライドイン

### 3. お題表示画面

#### レイアウト
- フルスクリーン、中央配置
- 背景に大きなグラデーション光源（上部から）

#### 要素
- **お題テキスト**: 画面中央、大きく表示
  - フォント: ディスプレイ、32-40px
  - 色: $warm-white
  - アニメーション: タイプライター効果で1文字ずつ出現
- **装飾**: お題を囲むようなネオンフレーム
- **進行ボタン**: 「数字を配る」

#### アニメーション
- お題テキスト: タイプライター効果（各文字50ms遅延）
- フレーム: フェードイン（お題表示完了後）
- 背景グロー: ゆっくりと脈動

### 4. 数字配布画面

#### 4-1. 「次の人に渡してください」画面
```
背景: $dark-bg（ほぼ真っ暗）
中央テキスト: 「{プレイヤー名}さんに渡してください」
サブテキスト: 「準備ができたらタップ」
アイコン: 回転する矢印またはハンドオーバーアイコン
```

#### 4-2. 数字表示画面
```
背景: ダークグラデーション + 中央に巨大なグロー
中央: 配布された数字（超大型、120px以上）
数字の装飾: 複数レイヤーのネオングロー、シャドウ
上部: 「{プレイヤー名}さんの数字」
下部: 「覚えたら次へ」ボタン
```

#### アニメーション
- 数字出現: スケールアップ（0 → 1.0）+ グロー拡大
- 3秒後に数字が少しずつぼやける（覚えてもらうための演出）
- ボタン: 遅延表示（数字表示から1秒後）

### 5. 表現タイム画面

#### レイアウト
- リラックスした雰囲気、圧迫感なし
- 中央に大きなテキスト

#### 要素
- **メインテキスト**: 「楽しく話し合ってね」
- **お題リマインダー**: 画面上部に小さくお題表示
- **装飾**: ゆっくり動くパーティクル or 光の粒
- **進行ボタン**: 「並び替えへ」

#### ムード
- 他の画面よりやや明るめのトーン
- パーティの楽しさを演出

### 6. 並び替え画面

#### レイアウト
- 上部: 説明テキスト「小さい順に並べてね」
- 中央: ドラッグ可能なプレイヤーカードリスト
- 下部: 「答え合わせ」ボタン

#### プレイヤーカード
```
サイズ: 幅100%、高さ64px
表示: 順位番号 + プレイヤー名
ドラッグ中: 浮遊感（シャドウ増加、スケール1.05）
ドロップ可能位置: ハイライト表示
```

#### インタラクション
- ロングプレスでドラッグ開始
- ドラッグ中は他カードが自動でスペースを空ける
- ドロップ時は滑らかにスロットイン
- 触覚フィードバック（可能であれば）

### 7. 答え合わせ＆結果画面

#### 7-1. 答え合わせアニメーション
```
各プレイヤーを順番に公開（1.5秒間隔）
公開時: カードがフリップ or スライドで数字表示
正解の場合: グリーンのハイライト + チェックマーク
不正解の場合: レッドのハイライト + ×マーク
```

#### 7-2. 最終結果
```
成功時:
- 背景: ゴールド系のグラデーション追加
- 大きな「SUCCESS!」テキスト + パーティクル演出
- 紙吹雪アニメーション

失敗時:
- 控えめな演出
- 「残念...」テキスト
- 正しい順番を表示
```

#### ボタン
- 「もう一度遊ぶ」: プライマリボタン
- 「最初から」: セカンダリボタン

## アニメーション仕様

### トランジション

#### 画面遷移
```
種類: フェード + スライド
時間: 300-400ms
イージング: cubic-bezier(0.4, 0.0, 0.2, 1)
```

#### 要素の出現
```
種類: フェードイン + 上からスライド（20px）
時間: 200-300ms
スタッガー: 各要素50ms遅延
```

### マイクロインタラクション

#### ボタンタップ
```
スケール: 1.0 → 0.95 → 1.0（100ms）
リップル: タップ位置から放射状に広がる
```

#### カードホバー/タップ
```
シャドウ増加
ボーダーグロー強化
微細なスケールアップ（1.02）
```

#### ネオングロー効果
```
@keyframes neon-pulse {
  0%, 100% { filter: drop-shadow(0 0 8px $neon-pink); }
  50% { filter: drop-shadow(0 0 20px $neon-pink); }
}
duration: 2s
iteration: infinite
```

## アクセシビリティ

### コントラスト比
- 本文テキスト: 4.5:1以上を確保
- 大きなテキスト: 3:1以上
- ネオン効果はグローを減らしてもテキストが読めること

### タッチターゲット
- 最小サイズ: 44 × 44 px
- ボタン間の間隔: 8px以上

### モーション軽減
- `prefers-reduced-motion`対応
- アニメーションを控えめにするオプション

## 実装時の注意点

### Flutterでの実装推奨
1. **アニメーション**: `flutter_animate`パッケージ推奨
2. **グロー効果**: `BoxShadow`を複数重ねて実現
3. **ドラッグ&ドロップ**: `ReorderableListView`または`flutter_reorderable_list`
4. **フォント**: Google Fontsパッケージ使用
5. **グラデーション**: `LinearGradient`、`RadialGradient`活用

### パフォーマンス考慮
- 過度なシャドウ・ブラーは控えめに
- アニメーションは`RepaintBoundary`で分離
- 画像アセットは適切に圧縮

## まとめ

**ピンク Ito**は、ダークモードをベースにネオンピンクのアクセントを効かせた、**大人向けのスタイリッシュなパーティゲームUI**を目指す。下ネタというカジュアルなテーマを、洗練されたビジュアルで包み込み、プレイヤーに「おしゃれなゲームで遊んでいる」という満足感を提供する。

記憶に残るポイント:
1. **ネオンピンクのグロー効果** - 全体を通じて統一感のあるアクセント
2. **滑らかなアニメーション** - 画面遷移、要素出現すべてに気を配る
3. **数字表示の演出** - ゲームのクライマックスを印象的に
4. **成功時の紙吹雪** - 達成感を視覚的に演出
